# Quiz Divertido

Arquivos principais:
- index.html
- style.css
- script.js
- questions.json

> Atenção: para que o `fetch("questions.json")` funcione, abra o projeto via **servidor local** (não use `file://`). Exemplos:

- **VS Code (Live Server)**: botão *Go Live* na barra de status.
- **Python**: `python -m http.server 5500` e acesse http://localhost:5500

## Pastas de assets
Crie as pastas e coloque seus arquivos:

```
img/
  - romano.png
  - egito.png
  - grecia.png
  - amazonas.png
  - galinha.png

sounds/
  - background.wav
  - correct.mp3
  - wrong.mp3
  - win.mp3
  - gameover.wav
```

Você pode alterar caminhos e nomes no `questions.json` e em `script.js`.
